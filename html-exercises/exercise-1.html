What is HTML?

HTML is the standard markup language for creating Web pages. HTML stands for Hyper Text Markup Language.
HTML describes the structure of a Web page.
HTML consists of a series of elements.

HTML elements tell the browser how to display the content. Write HTML using
Notepad or TextEdit. Web pages can be created and modified by using
professional HTMl editors.

However, for learning HTML we recommend a simple text editor like Notepad (PC) or TextEdit (Mac).
